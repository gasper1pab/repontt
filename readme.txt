Se usó visual studio code, versión 1.80.1
Se usó Cypress versión 9.5, con Node.js versión 14

Para uso de cypress, se puede seguir la guía : https://docs.cypress.io/guides/getting-started/installing-cypress

Una vez con los requisitos previos listo, abrir visual code y con el comando npx cypress run --spec 'saucedemo-pa', se puede correr directamente el js en el terminal, donde mostrará el resultado del ejercicio
O se lo puede ver gráficamente con el comando npx cypress open, una vez abierto el integrador de cypress, y seleccionando el archivo solicitado y seleccionar el navegador que se desee
